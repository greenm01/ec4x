Key Architecture Notes

  • Spacelift Ships: Per operations.md:1036, spacelift ships (ETAC, TroopTransport) are "individual units" NOT squadrons. They travel with fleets but are separate from combat squadrons.
  • Combat Phases: Space Combat → Starbase Assault → Ground Combat. Spacelift ships are screened during phases 1-2, participate in phase 3.
  • Task Forces: Temporary combat formations created when fleets converge in a system. All house fleets disband into squadrons that fight individually.
  • Fleet Structure: Fleet → Squadrons (combat) + SpaceLiftShips (transport/colonization)
