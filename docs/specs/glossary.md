# 10.0 Glossary & Definitions

| Term                  | Definition                                                                 | Section Link |
|-----------------------|----------------------------------------------------------------------------|--------------|
| AS                    | Attack Strength – offensive firepower of a ship or squadron                | 7.1.2       |
| CC                    | Command Cost – the "size" a ship takes in a squadron                       | 2.3.3       |
| CER                   | Combat Effectiveness Rating – multiplier applied to AS in Phases 1 & 3    | 7.3.3       |
| CL                    | Carry Limit – number of FS (fighters) or MD/PTU a carrier/transport holds  | 2.4.1 / 9.3 |
| CLK                   | Cloaking tech level                                                        | 2.4.3       |
| CR                    | Command Rating – maximum total CC a flagship can command                   | 2.3.3       |
| CST                   | Construction tech level – required shipyard level to build a ship class   | 9.1         |
| DS                    | Defense Strength – hits required to cripple or destroy a unit             | 7.1.2       |
| EBP                   | Espionage Budget Point – cost 40 PP each; used for covert actions          | 8.2         |
| ELI                   | Electronic Intelligence – detection tech level                            | 2.4.2       |
| ETAC                  | Environmental Transformation And Colonization ship – plants new colonies  | 2.3.2.3     |
| FD                    | Fighter Doctrine tech level – multiplier to per-colony fighter capacity   | 2.4.1       |
| FS                    | Fighter Squadron                                                           | 2.4.1       |
| GCO                   | Gross Colony Output – raw production before tax                           | 3.1         |
| IU                    | Industrial Units – production facilities on a colony                      | 3.5         |
| MD                    | Marine Division – ground invasion unit                                     | 2.4.9       |
| PC                    | Production Cost – build cost in PP                                         | 9.1–9.3     |
| PP                    | Production Points – currency of the game                                   | 3.0         |
| PTU                   | Population Transfer Unit – colonists moved by ETAC                         | 2.3.2.3     |
| PU                    | Population Units – inhabitants of a colony; basis for most limits         | 2.2         |
| ROE                   | Rules of Engagement – aggression level 00–10                               | 7.1.1       |
| SRP                   | Science Research Points – used to advance tech levels                      | 3.7         |
| Task Force            | Temporary merger of all fleets + starbase in a system during combat       | 7.2         |
